Clone down this repository. You will need `node` and `npm` installed globally on your machine.

ENV:

`MONGO_URI` && `JWT_SECRET`

Installation:

`npm install` && `cd client` && `npm install`

To Start Server:

`npm run dev`

To Visit App:

`localhost:3000/`

OR You can visit on

[Contact-Keeper](https://contact-keeper-test.herokuapp.com/)
